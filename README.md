# Swastik's Dev Hub

A cinematic, interactive digital portfolio experience for Swastik Bhardwaj, focusing on high-end animations and a premium developer aesthetic.

## Tech Stack

- **Framework**: React with TypeScript
- **Bundler**: Vite
- **Styling**: Tailwind CSS
- **Animations**: Framer Motion
- **UI Components**: shadcn/ui
- **Icons**: Lucide React

## Local Development

1. Clone the repository
2. Install dependencies:
   ```sh
   npm install
   ```
3. Start the development server:
   ```sh
   npm run dev
   ```

## Features

- **Hero Digital Universe**: Interactive background with cursor-driven shifts.
- **Interactive Skills Galaxy**: Rotating modules with pulse and hover effects.
- **Featured Projects Lab**: Cinematic project cards with animations.
- **Dev Console**: Terminal-style interface for a unique interaction experience.
